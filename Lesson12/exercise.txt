<h3>Lesson 12, Exercise</h3>
<p>We have a heading element,&nbsp;<code>.form__heading</code>, that we are manually assigning a color value to:&nbsp;<code>#fff</code>. Let&rsquo;s harness the power of Sass and have it generate a heading color based on the primary color of the page,&nbsp;<code>$color-primary</code>.</p>
<p>Among the functions available to us out of the box in Sass is the&nbsp;<code>adjust-hue()</code>&nbsp;function, which accepts two arguments. The first argument is the color that we want to adjust, and the second are the number of degrees that we want to shift the hue. There are 360 degrees on the color wheel, so a shift of 360 degrees would yield the identical color, and a shift of 180 degrees would produce the color&rsquo;s complement. For more on&nbsp;<code>adjust-hue()</code>, check out the documentation at&nbsp;<a href="http://sass-lang.com/documentation/Sass/Script/Functions.html#adjust_hue-instance_method" target="_blank" rel="noopener noreferrer">http://sass-lang.com/documentation/Sass/Script/Functions.html#adjust_hue-instance_method</a></p>
<p>Let&rsquo;s use adjust-hue() set the color of&nbsp;<code>.form__heading</code>, using&nbsp;<code>$color-primary</code>&nbsp;as its starting point.</p>
<p>The .scss file is located in /sass/main.scss. It will compile to /public/css/style.css.</p>
<p><strong>Tasks</strong></p>
<ul>
<li>Replace the&nbsp;<code>#fff</code>&nbsp;value for the color property of&nbsp;<code>.form__heading</code>&nbsp;with the&nbsp;<code>adjust-hue()</code>&nbsp;function</li>
<li>Pass&nbsp;<code>$color-primary</code>&nbsp;as&nbsp;<code>adjust-hue()</code>&rsquo;s first argument</li>
<li>Add a second argument for the amount that you want to shift the hue, which is completely up to you</li>
<li>Review the rendered page and the resulting color from our&nbsp;<code>adjust-hue()</code>&nbsp;operation</li>
<li>Review the CSS. The compiled value for&nbsp;<code>.form__heading</code>&rsquo;s color property is the result of the&nbsp;<code>adjust-hue()</code>&nbsp;operation</li>
<li>Adjust the value assigned to&nbsp;<code>$color-primary</code>&nbsp;to any of your choosing</li>
</ul>
