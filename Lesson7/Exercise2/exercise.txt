<h3>Lesson 7, Exercise 2</h3>
<p>We have a standard button, which is styled by the&nbsp;<code>.btn</code>&nbsp;class, which features a mint-green&nbsp;<code>background-color</code>. We also have a modified button, with a transparent background and mint-green border and font color, created through the&nbsp;<code>.btn--outline</code>&nbsp;modifier.</p>
<p>Now we want to make a color variant on those buttons, where&nbsp;<code>.btn--pink</code>&nbsp;sets the background of&nbsp;<code>.btn</code>&nbsp;to a pink color. And when we use also want to make an alternative to&nbsp;<code>.btn--outline</code>&nbsp;where the the border and font color are pink, rather than the stock mint-green. But we don&rsquo;t want to create an entirely separate modifier that creates both the outline aesthetic of the button, as well as the pink hue.</p>
<p>Instead we want to add&nbsp;<code>.btn--pink</code>&nbsp;to an element that already has a class of&nbsp;<code>.btn--outline</code>, and have it be both outlined AND pink.</p>
<p>The .scss file is located in /sass/main.scss. It will compile to /public/css/style.css.</p>
<p><strong>Tasks</strong></p>
<ul>
<li>Use nesting and the ampersand to create a&nbsp;<code>.btn--pink</code>&nbsp;modifier</li>
<li>Change the background color of&nbsp;<code>.btn--pink</code>&nbsp;to&nbsp;<code>#EA526F</code></li>
<li>Within the&nbsp;<code>.btn--outline</code>&nbsp;branch of the&nbsp;<code>.btn</code>&nbsp;block, nest a new selector, using the ampersand to join it with&nbsp;<code>.btn--pink</code></li>
<li>Within&nbsp;<code>.btn--outline.btn--pink</code>&nbsp;change the border color and font color to&nbsp;<code>#EA526F</code></li>
<li>Review the resulting HTML. There should be one solid mint-green button, one outlined mint-green button, one solid pink button, and one outlined pink button</li>
<li>Review the compiled CSS. There should be a&nbsp;<code>.btn selector</code>, a&nbsp;<code>.btn--pink</code>&nbsp;selector, a&nbsp;<code>.btn--outline</code>&nbsp;selector, and a&nbsp;<code>.btn--outline.btn--pink</code>&nbsp;selector</li>
</ul>
