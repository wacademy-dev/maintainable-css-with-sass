<h3>Lesson 8, Exercise</h3>
<p>Our buttons are either mint-green or pink, which means that we are using those two colour values a bunch of times. Let&rsquo;s clean things up and replace them with their own variables.</p>
<p>The .scss file is located in /sass/main.scss. It will compile to /public/css/style.css.</p>
<p><strong>Tasks</strong></p>
<ul>
<li>Create a variable to store the primary, mint-green color&nbsp;<code>#15DEA5</code>, and call it&nbsp;<code>$color-primary</code></li>
<li>Create a variable to store the secondary, pink color&nbsp;<code>#EA526F</code>, naming it&nbsp;<code>$color-secondary</code></li>
<li>Replace all instances of&nbsp;<code>#15DEA5</code>&nbsp;with the new&nbsp;<code>$color-primary</code>&nbsp;variable</li>
<li>Replace all instance of&nbsp;<code>#EA526F</code>&nbsp;with the new&nbsp;<code>$color-secondary</code>&nbsp;variable</li>
<li>Refresh the page and confirm that nothing has changed, visually speaking</li>
<li>Update the color value stored within&nbsp;<code>$color-primary</code>&nbsp;to one of your choosing.</li>
<li>Refresh the page to see that the first button now sports your updated color.</li>
<li>Review the CSS. Sass has replaced the variables with their hex values upon compiling</li>
</ul>
